# MOD3 project:
Using clustering algorithms to assess the validity of biotope codes assigned to plant surveys
## Task:
Application of three different clustering algorithms using GitHub during R project development
## Expected outcome:
short report containing three figures only (MOD3_project_report_Daniel_Pittner.pdf)
## Repository structure
This repository contains the three R files used in the project:
-	01_main_analysis.R contains the main analyses and results presented in the report
-	02_data_preparation.R includes data preparation steps and initial exploratory analyses as well as further testing
-	both files rely on functions defined in 03_functions.R
The results folder contains .RDS objects that are imported by scripts 01 and 02 to reduce runtime. By uncommenting the respective previous lines, the objects can also be recreated directly from the code. In addition, most figures generated by the scripts are stored in the results folder and are not overwritten when the code is run.
To execute the scripts 01 and 02, the working directory must be set to the source file location.
